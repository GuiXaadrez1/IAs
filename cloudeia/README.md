# Introdução

Este repositório tem como principal objetivo realizar um estudo de IA, mais específico o Cloud Code! 

## O que é o Claude IA? 

O Claude AI é uma família de grandes modelos de linguagem (LLMs) e um chatbot de inteligência artificial generativa criado pela Anthropic. é reconhecido por sua capacidade multimodal avançada, o que significa que ele processa com precisão textos, imagens, áudios e até interações diretas com interfaces de computador.A principal característica que define o Claude é a sua "personalidade" mais humana e cautelosa. Em linhas gerais, ele foi treinado para ser útil, honesto e inofensivo, utilizando uma técnica exclusiva chamada IA constitucional. 

## O que é o Claude Cowork?

Claude Cowork é um sistema baseado em agentes de IA. Diferente de chatbots tradicionais, ele não apenas responde perguntas, mas executa tarefas de forma autônoma no computador do usuário. A proposta é simples, transformar a IA em uma espécie de assistente digital ativo, capaz de organizar agendas, lidar com arquivos, automatizar processos e ajudar na execução de atividades profissionais, de acordo com o Time. A ferramenta utiliza o modelo Opus 4.5 ou +, uma versão mais avançada voltada para lidar com múltiplas tarefas ao mesmo tempo. A ideia central é reduzir a necessidade de intervenção humana em atividades repetitivas ou operacionais.

## O que é o Claude Code? 

Claude Code é como ter um ajudante muito esperto que vive dentro do seu terminal (aquela janela de texto preta que os desenvolvedores usam).

Você diz o que quer em palavras simples — "crie um site", "adicione uma página de login", "o que esse código faz?" — e ele vai lá e faz. Ele lê seus arquivos, escreve código, executa comandos e até testa as coisas, tudo por conta própria.

Pense assim: o Claude normal (no chat) é como mandar mensagem para um amigo inteligente. O Claude Code é como esse mesmo amigo sentado no seu computador, com as mãos no teclado, realmente fazendo o trabalho enquanto você assiste e aprova.

As principais diferenças em relação ao chat do Claude:

ele roda no seu terminal, não em um navegador

ele pode ver todo o seu projeto — todos os arquivos, o histórico do git, tudo

ele pode executar comandos — instalar pacotes, rodar testes, fazer commits de código

você permanece no controle — ele pergunta antes de fazer qualquer coisa destrutiva

Então, em vez de copiar código de uma janela de chat e colar no seu editor, o Claude Code simplesmente... faz isso diretamente.

É basicamente isso. Um colega de programação que vive no seu terminal.​​​​​​​​​​​​​​​​ Eu escrevi isso com o Claude.

### DIFERENÇA ENTRE COWORK E CLAUDE CODE

O Claude Code foi desenvolvido com foco em programadores e usuários avançados. Seu acesso acontece por meio de terminal, o que limita o uso ao público técnico.

Já o Claude Cowork aposta em uma interface mais simples e visual. A intenção é eliminar barreiras e permitir que qualquer pessoa utilize agentes de IA no cotidiano.

Essa mudança marca uma tentativa clara de popularizar o uso de sistemas autônomos, ampliando o alcance além do nicho de desenvolvedores.

## Claude Desktop

O Claude Desktop é um aplicativo nativo para macOS e Windows que coloca todo o poder do Claude diretamente no seu computador. Ele transforma o Claude de um simples parceiro de chat em um assistente ativo que está ali mesmo onde você trabalha. E o melhor: é gratuito para baixar, independente do seu plano

Dentro do Desktop você tem 3 ferramentas em uma só:

| Ferramenta      | O que faz                                                                 |
|----------------|--------------------------------------------------------------------------|
| 💬 Chat        | Conversa normal, igual ao navegador                                      |
| 🤖 Cowork      | Agente que executa tarefas no seu PC (organiza arquivos, gera documentos, etc.) |
| 💻 Claude Code | Integração com terminal e IDEs para programação                          |             |

## Tipos de Serviços e seus Focos:

### ⚡ Haiku — O Velocista

Ideal para: alto volume, tempo real, baixo custo

O Claude Haiku 4.5 é ultra-rápido e otimizado para respostas em tempo real e eficiência de custo, com preços de $1 por milhão de tokens de entrada e $5 por milhão de tokens de saída. Data Hackers

Na prática, o Haiku 4.5 chega a 80–120 tokens por segundo, com uma resposta de 500 palavras em apenas 2–3 segundos. Tech Insider

Casos de uso: chatbots de atendimento, moderação de conteúdo, automações simples, respostas rápidas em apps.

### ⚖️ Sonnet — O Equilibrado

Ideal para: uso profissional geral, melhor custo-benefício

O Claude Sonnet 4 é uma opção premium com custo de $3 por milhão de entrada e $15 por milhão de saída, compensando com desempenho superior em tarefas que exigem análise complexa. 

Data Hackers

O Sonnet 4.6 gera tokens a aproximadamente 40–60 tokens por segundo, com respostas de 500 palavras em cerca de 4–5 segundos — o dobro da velocidade do Opus para tarefas cotidianas. Tech Insider

Casos de uso: redação de e-mails e relatórios, análise de dados, desenvolvimento de software, suporte técnico, automação de fluxos de trabalho.

### 🏆 Opus — O Campeão Pesado

Ideal para: tarefas complexas, raciocínio profundo, agentes autônomos

O Claude Opus 4.7 é o modelo mais capaz disponível atualmente, com melhoria significativa em codificação de agentes em relação ao Opus 4.6. Claude API Docs

O Opus 4.6 tem um score de 80.8% no SWE-bench Verified, colocando-o entre os melhores modelos do mundo para código real. Tech Insider O preço é de $25 por milhão de tokens de saída.

Casos de uso: modelagem financeira, pesquisa científica, codificação avançada, agentes de IA autônomos, análise de documentos densos.
